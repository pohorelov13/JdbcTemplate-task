1. Реалізував клас Product який складається з: id, назва, ціна
2. Реалізував клас Cart, який складається зі списку продуктів доданих до цього кошика
3. Реалізував клас ProductDao за допомогою операції: видалення, додавання, пошук по id, отримання доступних продуктів. Даний клас взаємодіє з БД
4. Реалізував клас СartDao за допомогою операції: видалення, додавання, пошук за id. Даний клас взаємодіє з БД
5. Створив БД за допомогою таблиць: Products, Carts
6. Створив JdbcTemplate для встановлення з'єднання з БД
7. Створив Mappers для приведення даних SQL-форми в Java-об'єкт
